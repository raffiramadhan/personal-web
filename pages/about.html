Tentang Website Ini

Bagaimana website ini dibuat dan apa tujuannya.

Tujuan Website

Website ini dibuat sebagai bagian dari latihan membuat web statis menggunakan HTML dan CSS. Tujuannya adalah untuk melatih kemampuan dasar dalam membangun tampilan web yang rapi, konsisten, dan menarik.

Teknologi Yang Digunakan

🌐 HTML5 Digunakan untuk membangun struktur dasar halaman web.
🎨 CSS3 Berfungsi untuk mempercantik tampilan dan mengatur layout halaman.
⚡ JavaScript Memberikan interaktivitas pada elemen web agar lebih dinamis.
📱 Responsive Design Membuat tampilan website tetap rapi di berbagai ukuran layar.
💫 Modern UI Desain berfokus pada warna hangat, font bersih, dan kesan minimalis.
